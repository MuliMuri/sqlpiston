SQLPiston
=========

SQLPiston is a low-level SQL library that builds parameterized SQL queries
through Python operator overloading. Write once, query everywhere — AST nodes
carry zero SQL knowledge; dialect-specific compilers translate the same AST
into the right SQL for each database.
